
# Basic Chatbot 🤖

## 📌 Project Description

This is a simple rule-based chatbot developed using Python.
The chatbot responds to predefined user inputs such as greetings, questions, and exit commands.

This project was created as part of the CodeAlpha Python Programming Internship.

---

## 🚀 Features

- User-friendly conversation
- Predefined chatbot responses
- Continuous interaction using loops
- Exit option using "bye"
- Simple console-based interface

---

## 🛠️ Technologies Used

- Python
- Conditional Statements (if-elif-else)
- Loops
- User Input Handling

---

## ▶️ How to Run the Project

1. Install Python on your system.
2. Download the project files.
3. Open terminal or command prompt in the project folder.
4. Run the following command:

python chatbot.py
