
# Stock Portfolio Tracker 📈

## 📌 Project Description

This is a simple Stock Portfolio Tracker developed using Python.  
The program allows users to enter stock names and quantities to calculate the total investment value.

This project was created as part of the CodeAlpha Python Programming Internship.

---

## 🚀 Features

- User input for stock names and quantity
- Calculates total investment value
- Uses predefined stock prices
- Saves result into a text file
- Simple console-based interface

---

## 🛠️ Technologies Used

- Python
- Dictionary
- File Handling

---

## ▶️ How to Run the Project

1. Install Python on your system.
2. Download the project files.
3. Open terminal or command prompt in the project folder.
4. Run the following command:


python portfolio.py
