
# Hangman Game 🎮

## 📌 Project Description

This is a simple text-based Hangman Game developed using Python.  
The player has to guess the hidden word one letter at a time before the maximum number of wrong guesses is reached.

This project was created as part of the CodeAlpha Python Programming Internship.

---

## 🚀 Features

- Random word selection
- User input handling
- 6 wrong guess limit
- Win/Loss conditions
- Simple console-based interface

---

## 🛠️ Technologies Used

- Python
- Random Module

---

## ▶️ How to Run the Project

1. Install Python on your system.
2. Download the project files.
3. Open terminal or command prompt in the project folder.
4. Run the following command:

```bash
python Hangman.py
````
